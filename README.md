# 💬 ChatApp

Клиент-серверное приложение для обмена сообщениями в реальном времени, написанное на **C# + WPF**.

---

## 📋 Описание

ChatApp — это WPF-приложение с архитектурой клиент-сервер на основе **TCP-сокетов**. Сервер принимает подключения от нескольких клиентов одновременно, используя многопоточность. Клиент предоставляет графический интерфейс для отправки и получения сообщений, а также отображает список онлайн-пользователей в реальном времени.

## ⚙️ Как это работает

### Сервер (`ServerCore.cs`)

- Слушает входящие TCP-подключения на указанном порту (по умолчанию **5000**)
- Каждый клиент обрабатывается в отдельном фоновом потоке
- При подключении клиент обязан отправить команду `/join USERNAME` — иначе соединение закрывается
- Сервер ведёт словарь подключённых клиентов (`Dictionary<string, ClientInfo>`) с защитой через `lock`
- Генерирует события: `OnClientConnected`, `OnClientDisconnected`, `OnMessageReceived`, `OnServerLog`

### Клиент (`ClientCore.cs`)

- Подключается к серверу по IP-адресу и порту, отправляет `/join USERNAME`
- Принимает сообщения в отдельном фоновом потоке
- Различает обычные сообщения и системные (префикс `SERVER:`)
- Генерирует события: `OnConnected`, `OnDisconnected`, `OnMessageReceived`, `OnSystemMessage`, `OnError`

### Протокол общения

| Команда | Кто отправляет | Описание |
|---|---|---|
| `/join USERNAME` | Клиент → Сервер | Подключение с указанием имени пользователя |
| `/leave` | Клиент → Сервер | Отключение от сервера |
| `/users` | Клиент → Сервер | Запросить список онлайн-пользователей |
| `/pm USERNAME TEXT` | Клиент → Сервер | Личное сообщение другому пользователю |
| `SERVER:Welcome USERNAME!` | Сервер → Клиент | Подтверждение успешного подключения |
| `SERVER:Online: u1, u2` | Сервер → Клиент | Список онлайн-пользователей |
| `SERVER:USERNAME joined/left` | Сервер → Все | Уведомление о входе/выходе пользователя |
| `[PM from USERNAME]: TEXT` | Сервер → Клиент | Входящее личное сообщение |

## 🖥️ Интерфейс

### Сервер (700×500)
- Поле ввода порта и кнопки **Start Server** / **Stop Server**
- Статус (Running / Stopped) и счётчик подключённых клиентов
- Лог событий в реальном времени (шрифт Consolas)
- Кнопка **Clear Log**

### Клиент (900×600)
- Панель подключения: IP, порт, имя пользователя, кнопки **Connect** / **Disconnect**
- Основная область сообщений с автопрокруткой
- Боковая панель **Online** со списком подключённых пользователей
- Поле ввода сообщения + кнопка **Send** (также работает клавиша **Enter**)

## 🚀 Запуск

### Через Visual Studio

1. Открыть `ChatApp.slnx`
2. Запустить **ChatServer** (правой кнопкой → Set as Startup Project → Start)
3. Запустить **ChatClient** (можно несколько экземпляров)

### Через командную строку

```bash
# Клонировать репозиторий
git clone https://github.com/CoBaLtKaMi/ChatApp.git
cd ChatApp

# Запустить сервер
cd ChatServer
dotnet run

# В другом терминале — запустить клиент (можно несколько)
cd ../ChatClient
dotnet run
```

### Подключение

1. Запустить сервер, указать порт (по умолчанию `5000`), нажать **Start Server**
2. В клиенте ввести IP сервера (`127.0.0.1` для локального), порт и имя пользователя
3. Нажать **Connect**
4. Начать общаться
